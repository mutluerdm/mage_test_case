## register
```sh
Post /v1/user/register
```
#### params

```javascript
{
  "username" : string,required,alphanum,min=3,max=10
  "password" : bool,required,alphanum,min=3,max=10
}
```

#### example response

```javascript
{
"status": "success",
"timestamp": 1661201326,
"result":{
  "id": 18,
  "username": "erdem12",
  "password": "5375Erdem"
  }
}
```
